# Tera Stream Pro

**Advanced Telegram Mini App Video Player** with AdsGram monetization.

## Features

- Elegant preview card with thumbnail & metadata
- Custom fullscreen player (seek, speed 0.5x-2x, skip, controls)
- **AdsGram Interstitial** before playing video
- HLS streaming support
- Mobile-optimized dark UI
- Direct download option

## Usage

1. Open Mini App in Telegram
2. Paste TeraBox link or use shared link
3. Click **Watch Now** → Watch short ad → Enjoy video

## Tech

- Vanilla HTML/CSS/JS
- AdsGram SDK (`int-37243`)
- Custom video controls

**Deploy** `index.html` on any static host and link via BotFather.

Made for seamless TeraBox experience!
