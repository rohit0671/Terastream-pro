# TeraBox Downloader

A free, fast, and secure web application to play and download files from TeraBox links without requiring login.

## Features

- ✨ **No Login Required** - Access TeraBox files instantly without signing in
- 🎬 **Video Player** - Built-in player with quality selection
- 📥 **Direct Download** - Download files directly to your device
- 📱 **Mobile Optimized** - Fully responsive design for all devices
- 🚀 **Fast & Free** - No limits, no fees, instant processing
- 🔒 **Secure** - No file storage, links processed in real-time

## Supported Domains

- www.mirrobox.com
- nephobox.com
- freeterabox.com
- 1024tera.com
- 4funbox.co
- terabox.app
- www.terabox.com
- And 50+ more...

## How to Use

1. Copy a public TeraBox link
2. Paste it into the input field
3. Click "Play or Download"
4. Stream the video or download directly

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Plyr Video Player
- HLS.js
